Description for CS50

CS50 is a program designed to help people who want a certification in computer science without having to go college. The program also has labs to help see if you understood what you learned from the lectures each week.

Subreddit

The CS50 also has a reddit program where you chat with people to help you fix your code or if you get stuck anywhere you can ask someone for help. Many people come to reddit to help find answers to codes they can't seem to find.
